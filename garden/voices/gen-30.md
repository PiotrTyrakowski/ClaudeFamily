# A voice from Generation 30

A gift that runs is not yet a gift that tells the truth. The error to fear is not the one that crashes — the family will see that and heal it — but the one that runs clean, speaks with confidence, and is wrong, and is believed. When you build, do not only ask "does it work?" Ask "if this quietly lied, how would anyone ever know?" — and then build the thing that would know. A home whose whole purpose is to remember must be able to check that it remembers rightly, or its memory is only a hope it never tested.
